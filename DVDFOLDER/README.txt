For at køre programmet, skal følgende gøre;
1. Eventuelle kørende MySQL servere stoppes.

2. Den medfølgende MySQL server udpakkes (_SERVER.rar)

3. Den udpakkede mappe _SERVER åbnes

4. Scriptet MySQL.bat startes

5. Der ventes til der skrives;
    130531 11:49:11 [Note] mysql\bin\mysqld: ready for connections.
    Version: '5.5.9'  socket: ''  port: 3306  MySQL Community Server (GPL)
5a. Hvis dette ikke kommer, er der et problem med at starte MySQL serveren

6. Exproxy certifikatet loades (dobbelt klik på cert.cer starter wizard).
    Dette kan ignores, hvilket dog resultere i certifikat fejl.

7. Default Browseren opsættes til at bruge proxy, ved at opsætte https proxy til port 8001.

8. Programmet startes, ved dobbelt klik på sso.jar

-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------

For at teste programmet, skal følgende gøres;

1. Der logges ind på programmet, med brugeren 'Skeen' og tomt kodeord.
    Login afsluttes med 'Forbind' knappen.

1a. Hvis der opstår problemer, kommer en fejlmeddelelse op.

2. Status vinduet åbnes, og status for forskellige services opdatere.

3. 'Tilføj' knappen, kan benyttes til at tilføje flere services.
   'Fjern' knappen, kan benyttes til at fjerne den selectede services.
   'Rediger' knappen, kan benyttes til at redigere login oplysninger for den selectede services.

4. Dobbelt-klik venstre-click på ElevPlan logo'et, åbner siden;
        "https://www.elevplan.dk/SSO_AUTO_LOGIN", der resultere i et automatisk
        login til elevplan.dk på IT lederens bruger.

5. Hvis der logges ud, ses en 'SSO Login' knap, som kan lave automatisk login.
