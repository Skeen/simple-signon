{\bf Backlog} Grundet projektets slutning, er disse ting ikke blevet
implementeret; 
\begin{itemize} 
\item Kryptering af key-value information, på
databasen.  
\item Integrering med Active Directory (login proces) \item
Integrering med Active Directory Domæne Server (bruger synkronisering) \item
Redesign eller fjernelse af 'Opdater' knappen \item Ordenlig injektion af HTML i
proxy svar (vha. jsoup) \item Ordenlig implementering af VPN support, og wifi
\item Status Checking af webservices \item Mulighed for disconnection af
webservices \item Mulighed for autogenerede genveje (short-cuts) på skrivebordet
\item Understøttelse af højre-click kontekst menu (på status vinduet) \item
Vedholdne computer specifikke konfigurationer, f.eks. vindue størrelse \item
Ydereligere refactoring og oprydning af source koden 
\end{itemize}
